## ``qcdenoise``

(Classical) Machine Learning-based error mitigation of gate-based quantum computers

### Install 
To install `qcdenoise`
```
git clone https://github.com/nlaanait/qcdenoise.git
cd qcdenoise
```
with `pip`
```
pip install .
```
alternatively,
```
python setup.py install
```
### Major Dependencies
1. `qiskit` (IBM)
